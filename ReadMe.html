Introduction:

 

Thank you for downloading FlexPDE, the premier software product for solving the coupled sets of partial differential equations encountered in the mathematical description of most physical phenomena.

 

Downloaded Version

 

The version of FlexPDE that you have downloaded is internally set to run in demonstration mode.  Only specially keyed problem scripts can be run with the full capability of FlexPDE.  Unkeyed scripts that you create will be restricted to five simultaneous equations and 100, 800 or 1600 mesh nodes in 1D, 2D and 3D, respectively. 

 

To unlock the full power of  FlexPDE you will need to purchase a license and receive from PDE Solutions the correct hardware or software license key.  See the license agreement displayed during your first invocation of FlexPDE, or click “Help | License” to redisplay the agreement. 

 

To purchase a license, go to our website (www.pdesolutions.com) and click “Buy Now”.  You can then proceed to our web store or download a FAX order form. You can also request a free 30-day trial of the software.

Documentation

 

FlexPDE includes an online Help system which fully describes the problem description process.  The contents of this help system are also included in hardcopy documentation included with purchase of a license.   The separate booklets are “Getting Started”, “User Guide” and  “Problem Descriptor Reference”.  “Getting Started” describes how to install the software and how to use the graphical user interface.  The “User Guide” is a tutorial guide through the use of the FlexPDE scripting language. “Problem Descriptor Reference” is a detailed description of the rules of the scripting language.

FlexPDE

 

Developed by the staff that originally developed PDEase2, FlexPDE is a unique, flexible, and powerful general purpose software system for obtaining numerical solutions to the coupled sets of partial differential equations frequently found in engineering, physics, chemistry, biology, geology, mathematics and other scientific fields.

 

FlexPDE is being used by customers both to teach Physics and to solve numerous industrial engineering and chemical design problems. The code modules from which FlexPDE is assembled are part of a library of code modules that PDE Solutions Inc. uses to create custom software for its customers.

 

FlexPDE uses the powerful finite element method to obtain its numerical solutions.  FlexPDE does not, however, require its users to have any knowledge of the intricacies of the finite element method or mesh generation.  Users who can express their problems as initial value/boundary value problems can use FlexPDE immediately.

 

For those customers that have difficulty in setting up their problem, PDE Solutions Inc. offers contract services to set up problems, provide numerical studies, or document methods and procedures.
